# Predicción de fallas en equipos industriales

## 📌 Descripción
Este proyecto analiza datos de mantenimiento industrial para identificar patrones y predecir fallas en equipos mediante Machine Learning.

## ⚙️ Tecnologías
- Python
- Pandas
- Matplotlib / Seaborn
- Scikit-learn

## 📊 Análisis realizado
- Limpieza de datos
- Análisis exploratorio
- Visualización
- Modelo predictivo (Regresión Logística)

## 🤖 Resultados
- Variables más influyentes: antigüedad, uso diario
- Modelo capaz de predecir fallas con buena precisión
